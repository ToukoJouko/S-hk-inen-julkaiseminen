CSS - suorakaidemalli

CSS-asettelu ja asioiden esitystapa perustuu suorakaiteen muotoisiin lohkoihin. Suorakaidemalli muistuttaa pitkälti HTML:stä tuttuja lohkoelementtejä. HTML-lohkoelementtejä ovat mm. taulukot, tekstikappaleet, otsikot, listat ja div-elementit.

Width

Width-ominaisuus ilmoittaa elementin leveyden. Arvoina on helpointa käyttää pikseleitä tai prosentteja. Annettaessa arvo prosentteina, määrittyy elementin koko suhteessa sitä ympäröivään elementtiin.

  • leveys annetaan aina lukuarvona
  • arvo määrittää ainoastaan elementin sisällön leveyden - täytteet ja marginaalit lisätään leveyteen
  • max-width-ominaisuudella voidaan määrittää enimmäisleveys
  • max-width on kätevä esim. tekstikappaleiden, koko sivun tai joustavan sisältöosan leveyden rajoittamiseen
  • min-width-ominaisuudella voidaan määrittää minimileveys, tarvitaan harvoin

Height

Height-ominaisuus ilmoittaa elementin korkeuden. Arvoina on helpointa käyttää pikseleitä tai prosentteja. Annettaessa arvo prosentteina, määrittyy elementin koko suhteessa sitä ympäröivään elementtiin. Useimmissa tapauksissa elementin korkeus on fiksua jättää määrittelemättä ja antaa elementin joustaa sisällön vaatiman tilan mukaan.

  • arvo määrittää ainoastaan elementin sisällön korkeuden - täytteet ja marginaalit lisätään korkeuteen
  • min-height-ominaisuudella voidaan määrittää elementin minimikorkeus
  • min-height voi olla hyödyllinen esimerkiksi sivun jonkin osion vähimmäiskorkeuden määrittämisessä

Margin

Kaikkien HTML-lohkoelementtien (tekstikappale, otsikkotaso, kuva, lista, taulukon solu jne.) ympärille (elementin ulkopuolelle) voidaan määritellä marginaali eli tyhjä tila. Marginaalit ovat aina läpinäkyviä. Marginaaleille voidaan antaa arvo pituusyksikköinä (esim. px) tai prosentteina. Jos yksikkönä käytetään prosentteja, lasketaan marginaalin koko suhteessa elementtiä ympäröivään tilaan (esim. selainikkunaan). Marginaalien arvojen antojärjestys on ylhäältä alkaen myötäpäivään, eli:

  • ylämarginaalin arvo
  • oikean marginaalin arvo
  • alamarginaalin arvo
  • vasemman marginaalin arvo

Jos kaikki marginaalit (ylä, oikea, ala, vasen) halutaan määrittää saman suuruisiksi, riittää että annetaan vain yksi arvo. Seuraava esimerkki määrittää kaikki tekstikappaleiden (p) marginaalit yhdellä kertaa 15 pikselin suuruisiksi.

p { margin: 15px; }

Seuraava esimerkki poistaa sivulta ylä- ja alamarginaalit mutta tekee sivun oikealle ja vasemmalle reunalle 8% levyiset marginaalit. Prosentteina annettavat marginaalit ovat suhteellisia, eli selainikkunan koon mukaan määrittyvinä.

body { margin: 0% 8% 0% 8%; }

Edellä kuvattua järjestystä (myötäpäivään ylhäältä alkaen) kannattaa selkeyden vuoksi noudattaa myös silloin kun jokainen marginaali määritellään erillisillä ominaisuus-arvo -pareilla. Seuraava esimerkki toteuttaa täsmälleen saman asian kuin edellinen, ainoastaan eri tavalla kirjoitettuna:

body {
  margin-top: 0%;
  margin-right: 8%;
  margin-bottom: 0%;
  margin-left: 8%;
}

Padding

Suorakaidemallissa kaikille lohkoelementeille voidaan määritellä täyte (padding). Täyte tarkoittaa elementin reunan ja sen sisällön väliin jäävää tyhjää tilaa, vaikkapa taulukkoon sijoitetun tekstin ja taulukon solun reunan välistä tyhjää tilaa. Jos padding ominaisuudelle annetaan vain yksi arvo, se määrittää täytteen leveyden elementin jokaiselle sivulle samaksi. Jos elementin eri sivuille halutaan antaa eri suuruiset täytteet, tulee jokaisen sivun täyte määritellä erikseen. Aivan kuten marginaaleja asetettaessa. Seuraava esimerkki määrittää taulukon solun (td) sisällän ja solun reunan väliin on 10 pikselin suuruisen täytteen:

td { padding: 10px; }

Seuraava esimerkki määrittelee taulukon solun täytteiksi 5 pikseliä ylös ja alas, ja 10 pikseliä oikealle ja vasemmalle:

td { padding: 5px 10px 5px 10px; }

Border

Suorakaidemallissa kaikki lohkoelementit varaavat itselleen suorakaiteen muotoisen tilan. Tilan rajoja kutsutaan reunuksiksi. Oletuksena elementtien reunukset ovat näkymättämiä mutta ne saadaan näkyviksi määrittämällä reunukselle väri, tyyppi ja paksuus.

  • border-width = reunaviivan leveys
  • border-color = reunaviivan väri
  • border-style = reunaviivan tyyppi

Millaiset reunukset seuraava esimerkki tekee ja mille?

table {
  border-width: 0px 1px 0px 1px;
  border-style: solid;
  border-color: red;
}
