CSS - värit ja taustavärit

Sivuston värejä määritettäessä kannattaa ehdottomasti käyttää kunnollista värihallintatyökalua. Kunnolliset CSS-editorit sisältävät värien valintaan tarkoitetun työkalun mutta verkosta löytyy myös erillisiä värinhallintatyökaluja. Eräs hyvä ja monipuolinen vaihtoehto on http://paletton.com. Esteettisten asioiden lisäksi värien määrittelyssä on syytä muistaa muutamia teknisiä seikkoja:

  • kun määrität elementin piirtovärin, määritä aina myös taustaväri (esim. tekstin värin + taustaväri)
  • jos määrität sivuillesi oletuksesta poikkeavan taustavärin, muista aina määrittää myös piirtoväri
  • älä käytä tekstin alla taustakuvia, ne heikentävät tekstin luettavuutta
  • jos muutat linkkien värejä, määritä aina myös linkkien taustavärit

Color

Color-ominaisuus asettaa elementin piirtovärin. Väri on color-ominaisuudessa mahdollista määrittää hyvin monilla eri arvoilla (teksti, heksadesimaali, kymmenjärjestelmä, prosentti). Värien määritys tapahtuu todellisuudessa kuitenkin aina RGB-järjestelmän mukaan heksadesimaaleina. Tämä johtuu siitä, että tietokoneet käsittelevät värejä aina RGB-järjestelmän mukaisesti. Tekstimääritystä, kymmenjärjestelmää ja prosenttimerkintää voidaankin ajatella vaihtoehtoisina esitystapoina heksadesimaaleille.

  • teksti: color: lime;
  • prosentti: color: rgb(0%,100%,0%);
  • kymmenjärjestelmä: color: rgb(0,255,0);
  • heksadesimaali: color: #00FF00;

RGB-värijärjestelmässä jokainen väri muodostetaan punaisen (R), vihreän (G) ja sinisen (B) sekoituksena. Kunkin värin osuus ilmoitetaan kahdella numerolla tai kirjaimella. RGB-heksadesimaali on siis aina kuuden luvun tai kirjaimen pituinen. Kuusinumeroinen heksadesimaali voidaan joissain tapauksissa lyhentää kolminumeroiseksi. Lyhentäminen on mahdollista, jos yksittäisen värin molemmat arvot ovat samat. Esimerkiksi #f0f tarkoittaa samaa kuin #ff00ff.

Suosittelen alusta asti totuttelemaan heksadesimaalien käyttöön. Heksadesimaaleja käytettäessä väriarvojen kopiointi esimerkiksi kuvankäsittelyohjelmasta helpottuu ja totut värihallintaohjelmien ja kuvankäsittelyohjelmien tapaan merkitä värejä.

  • heksadesimaaliarvo alkaa CSS-tiedostossa aina risuaitamerkillä: #
  • heksadesimaali koostuu kuudesta peräkkäisestä numero ja/tai kirjainparista (Red, Green, Blue)
  • kaksi ensimmäistä lukua/kirjainta ilmoittaa punaisen määrän
  • kaksi keskimmäistä lukua/kirjainta ilmoittavat vihreän määrän
  • kaksi viimeistä lukua/kirjainta ilmoittavat sinisen määrän
  • jos värin molemmat luvut/kirjaimet ovat samat, se voidaan lyhentää yhdellä luvulla/kirjaimella esitettäväksi

Seuraava esimerkki määrittää tekstikappaleen piirtovärin. Punaisen, vihreän ja sinisen arvo on 00. Tuloksena on musta.

p { color: #000000; }

Seuraava esimerkki määrittää taulukon soluun sijoitetun tekstikappaleen piirtovärin. Punaisen, vihreän ja sinisen arvo on 66. Tuloksena on harmaa.

td p { color: #666666; }

Background-color

Background-color-ominaisuus määrittää elementin taustavärin. Mille tahansa elementille voidaan määrittää taustaväri. Seuraava esimerkki asettaa koko dokumentin taustaväriksi hopeanharmaan.

body { background-color: #F3F3F3; }

Tekstikappaleen piirtoväriksi esitetään tummanharmaata ja taustaksi valkoista seuraavasti.

p {
  color: #474747;
  background-color: #FFFFFF;
}

Monissa tapauksissa on kätevää määrittää elementtien taustaväri läpinäkyväksi. Tällöin elementin taustalla olevia värejä voidaan muuttaa ilman että päällä olevan elementin taustaväriä tarvitsee määrittää uudestaan. Läpinäkyvä taustaväri saadaan antamalla arvoksi transparent.

h1 {
  color: #474747;
  background-color: transparent;
}
